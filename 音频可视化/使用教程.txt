===========================================================
          音频可视化工具 —— 使用教程
===========================================================

【安装依赖】pip install pydub Pillow
【还需要】系统安装 FFmpeg（用于读取 MP3 格式）
【运行】双击 AudioVisualizer.py


===========================================================
  五种可视化风格
===========================================================

  1. 经典波形  → 上下起伏的连续曲线，最经典的波形图
  2. 柱状波形  → 一根根竖条，适合做封面
  3. 镜像波形  → 上下对称，上半实色下半淡影，最好看
  4. 渐变山脊  → 从底部生长的山脉效果，渐变填充
  5. 频谱色带  → 用彩虹色映射音量大小，最炫


===========================================================
  八种配色方案
===========================================================

  霓虹蓝紫  → 蓝 → 紫 → 粉，暗底（默认）
  日落橙红  → 金 → 橙 → 红，暖色调
  森林绿    → 亮绿 → 深绿，自然感
  海洋青蓝  → 青 → 蓝，冷色调
  樱花粉    → 粉 → 桃红，温柔感
  纯白极简  → 白底灰色波形，干净简约
  赛博黄绿  → 黑底绿光，黑客/赛博朋克风
  深空灰橙  → 深灰底橙色，稳重


===========================================================
  操作步骤
===========================================================

  1.「添加音频」选择 MP3/WAV/FLAC/OGG 等音频文件
     也可以「添加文件夹」一次导入整个文件夹

  2. 选择可视化风格和配色方案

  3. 设置输出尺寸（或点预设快捷按钮）

  4. 三种操作方式：
     - 预览第一个 → 快速看效果
     - 单个保存   → 渲染第一个并保存
     - 批量导出   → 所有音频全部渲染并保存


===========================================================
  输出尺寸预设
===========================================================

  手机壁纸  → 1080 x 1920（竖屏）
  电脑壁纸  → 1920 x 1080（横屏）
  社交封面  → 1200 x 630（微信/微博封面）
  正方形    → 1080 x 1080（朋友圈/Instagram）

  也可以自定义任意尺寸


===========================================================
  其他参数
===========================================================

  【波形精度】
    自动 → 根据图片宽度自动计算
    手动 → 100~800，数字越大波形越细腻

  【显示时长/采样率】
    勾选后图片底部显示时长、采样率、声道信息

  【显示文件名】
    勾选后图片顶部显示音频文件名


===========================================================
  支持的音频格式
===========================================================

  MP3、WAV、FLAC、OGG、M4A、AAC、WMA
  （需要系统安装 FFmpeg）


===========================================================
  适合用来做什么
===========================================================

  - 音乐封面 / 专辑封面
  - 播客节目封面
  - 社交媒体音乐分享配图
  - 视频剪辑的音频可视化素材
  - 手机/电脑壁纸
  - 博客/公众号配图
