===========================================================
          ASCII 艺术生成器 —— 使用教程
===========================================================

【安装依赖】pip install Pillow
【运行】双击 ASCIIArt.py


===========================================================
  两种模式
===========================================================

  ┌──────────────┬──────────────────────────┐
  │ 图片 → ASCII │ 把照片/图片变成字符画    │
  │ 文字 → 大字  │ 把文字变成 ASCII 大字横幅│
  └──────────────┴──────────────────────────┘


===========================================================
  模式 1：图片 → ASCII
===========================================================

  1. 点「选择图片」加载一张图片
  2. 设置参数：
     - 输出宽度（字符数，默认 100）
     - 字符集（标准/详细/方块/简约/自定义）
     - 反转（勾选后适合白色背景显示）
  3. 点「生成 ASCII 艺术」

  字符集说明：
    标准 →  .:-=+*#%@         10 级灰度
    详细 →  .'`:;Il!i><~ ...  70 级灰度（最细腻）
    方块 →  ░▒▓█              色块风格
    简约 →  .oO@#             5 级
    自定义 → 自己填写字符（从暗到亮排列）


===========================================================
  模式 2：文字 → 大字
===========================================================

  1. 切换到「文字 → 大字」标签
  2. 输入英文/数字（支持 A-Z 0-9 和常用符号）
  3. 设置填充字符（默认 #，可换成 * @ █ 等）
  4. 设置字间距
  5. 点「生成大字」

  效果示例：
    #   #  #####  #      #      #####
    #   #  #      #      #      #   #
    #####  ####   #      #      #   #
    #   #  #      #      #      #   #
    #   #  #####  #####  #####  #####


===========================================================
  导出方式
===========================================================

  复制到剪贴板 → 粘贴到聊天/邮件/代码注释
  保存为 TXT   → 纯文本文件
  保存为图片   → 渲染成绿色字符风格 PNG（黑底绿字）
