===========================================================
          代码截图美化 —— 使用教程
===========================================================

【安装依赖】pip install Pillow pygments
【运行】双击 CodeScreenshot.py


===========================================================
  使用方法
===========================================================

  1. 在左侧文本框粘贴你的代码
  2. 在右侧选择语言、主题、背景等参数
  3. 点「生成预览」查看效果
  4. 满意后点「保存图片」导出 PNG/JPG


===========================================================
  参数说明
===========================================================

  【语言】
    选择代码语言（Python、JavaScript、Java 等 20+）
    选对语言才能正确高亮关键字

  【代码主题】
    Pygments 内置 20+ 款配色主题：
    - monokai    → 经典深色（推荐）
    - dracula    → 暗紫主题
    - one-dark   → Atom 风格
    - github-dark → GitHub 深色
    - solarized-light → 浅色护眼

  【背景】
    代码窗口外面的背景色：
    - 紫蓝渐变、蓝绿渐变、橙红渐变（适合分享）
    - 深空灰（低调）
    - 透明（保存为 PNG 可直接叠加）

  【字号】
    代码字体大小（10-30），默认 16

  【内边距】
    代码窗口的内边距，默认 40px

  【窗口圆点】
    macOS 风格的红黄绿三个圆点（可关闭）

  【行号】
    是否在左侧显示代码行号

  【窗口标题】
    可选的标题文字（如 main.py）


===========================================================
  导出说明
===========================================================

  PNG → 无损，支持透明背景
  JPG → 体积更小，白底

  输出图片是高清的，直接用于：
  - 朋友圈/微博分享
  - 技术博客配图
  - PPT/Keynote 插图
  - 打印
