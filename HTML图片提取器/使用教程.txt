===========================================================
         HTML 图片提取器 —— 使用教程
===========================================================

【安装依赖】
    无需安装任何依赖，直接运行

【运行】
    双击 HTMLImageExtractor.py 或命令行 python HTMLImageExtractor.py


【操作步骤】
1. 选择导入方式（三选一）：
   - 「选择 HTML 文件」→ 选择一个或多个 .html 文件
   - 「从剪贴板粘贴 HTML 代码」→ 直接粘贴 HTML 源码
   - 「选择文件夹（批量扫描）」→ 扫描整个文件夹的所有 HTML
2. 程序自动解析，列表显示找到的所有图片
3. 查看列表：文件名、格式、来源类型、大小、标签信息
4. 可用筛选下拉框过滤：仅 PNG / 仅 SVG / 仅 Base64 等
5. 选中要导出的图片（Ctrl 多选），点「导出选中图片」
   或直接点「导出全部」一键保存所有图片
6. 选择保存文件夹，完成后自动打开目录


【能提取的图片类型】

  Base64 内嵌图
    → HTML 里的 src="data:image/png;base64,..." 编码图片
    → 自动解码还原为 PNG/JPG/GIF 等原始文件

  <img> 标签引用
    → <img src="logo.png"> 引用的本地或网络图片
    → 本地文件直接复制，网络图片可选下载

  内联 SVG
    → 直接写在 HTML 里的 <svg>...</svg> 矢量图
    → 提取保存为 .svg 文件

  CSS 背景图
    → style 或 <style> 中的 background-image: url(...)
    → 支持本地和网络引用

  Favicon 图标
    → <link rel="icon" href="favicon.ico"> 网页图标
    → <link rel="apple-touch-icon"> 苹果触屏图标

  其他
    → <video poster="..."> 视频封面图
    → <source srcset="..."> 响应式图片
    → <input type="image"> 图片按钮


【筛选功能】
  底部下拉框可以按条件筛选：
  - 全部 / 仅 PNG / 仅 JPG / 仅 SVG / 仅 ICO
  - 仅 Base64（只看内嵌图）
  - 仅本地文件（只看引用的本地图片）
  - 仅网络资源（只看引用的网络图片）


【网络图片下载】
  勾选「下载网络图片」→ 导出时自动下载网络引用的图片
  不勾选 → 跳过网络图片，只导出本地和 Base64 图片


【小技巧】
  - 按住 Ctrl 点击可多选，Shift 点击可连选
  - 点「全选」再点「导出全部」最方便
  - 批量扫描文件夹时会递归搜索所有子目录
  - 支持 .html .htm .xhtml 格式
