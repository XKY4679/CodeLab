===========================================================
          屏幕取色器 —— 使用教程
===========================================================

【安装依赖】pip install Pillow
【运行】双击 ColorPicker.py


===========================================================
  使用方法
===========================================================

  1. 打开工具，点「开始取色 (F1)」进入取色模式
  2. 移动鼠标到屏幕任意位置 → 实时显示颜色
  3. 按 空格 或 回车 → 锁定当前颜色并自动复制 HEX
  4. 按 Esc 或点按钮 → 停止取色

  窗口始终置顶，不影响操作其他软件


===========================================================
  界面说明
===========================================================

  左上：颜色预览大方块（显示 HEX 值）
  右上：放大镜（10 倍放大，红色十字准心定位）
  中间：四种格式的颜色值（点击即复制）
        HEX   → #FF5733
        RGB   → rgb(255, 87, 51)
        HSL   → hsl(11, 100%, 60%)
        RGBA  → rgba(255, 87, 51, 1.0)
  下方：历史记录（点击恢复 / 右键删除）


===========================================================
  快捷键
===========================================================

  F1      → 开始/停止取色
  空格    → 锁定当前颜色
  回车    → 锁定当前颜色
  Esc     → 停止取色
