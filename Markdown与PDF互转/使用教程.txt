===========================================================
        Markdown 与 PDF 互转 —— 使用教程
===========================================================

【安装依赖】
    pip install fpdf2 pdfplumber

【运行】
    双击 MarkdownPDF.py 或命令行 python MarkdownPDF.py


===========================================================
  标签页一：Markdown → PDF
===========================================================

【操作步骤】
1. 切换到「Markdown → PDF」标签页
2. 点「选择 .md 文件」，选一个 Markdown 文件
3. 预览区会显示文件内容，确认是否正确
4. 点「转换为 PDF」→ 选择保存位置
5. 转换完成，打开 PDF 查看效果

【支持的 Markdown 语法】
  # 一级标题          → 大号粗体 + 下划线
  ## 二级标题         → 次大粗体 + 下划线
  ### 三级标题        → 中号粗体
  #### 四级标题       → 小号粗体
  **加粗文字**        → 加粗显示
  *斜体文字*          → 斜体显示
  - 无序列表          → 圆点列表
  1. 有序列表         → 数字列表
  > 引用文字          → 带左侧竖线的引用块
  ```代码块```        → 灰色背景代码区域
  ---                 → 水平分割线
  [链接](url)         → 显示链接文字
  ![图片](path)       → 显示 [图片]

【中文支持】
  自动使用微软雅黑字体渲染中文，无需额外配置


===========================================================
  标签页二：PDF → Markdown
===========================================================

【操作步骤】
1. 切换到「PDF → Markdown」标签页
2. 点「选择 PDF 文件」，选一个 PDF
3. 点「提取文本」→ 程序自动分析 PDF 内容
4. 提取结果显示在文本框中，可以手动编辑调整
5. 满意后点「保存为 .md」保存为 Markdown 文件
6. 也可以点「复制全部」直接粘贴到其他地方

【智能识别】
  - 自动根据字号大小识别标题（大字 → # 标题）
  - 每页之间用 --- 分隔
  - 识别结果可能不完美，建议手动调整后再保存

【注意事项】
  - PDF 转 Markdown 是「尽力提取」，不是 100% 还原
  - 扫描版 PDF（图片型）无法提取文字
  - 复杂排版（多栏、表格）可能需要手动调整
  - 提取后的文本框可以自由编辑，调整到满意再保存
